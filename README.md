# プロジェクト別タスク管理アプリ

## 概要
プロジェクトごとにタスクを管理できるWebアプリケーションです。タスクの登録・編集・削除・完了処理などをプロジェクト単位で行えます。

## 使用技術
- **バックエンド**: FuelPHP  
- **フロントエンド**: Knockout.js  
- **データベース**: MySQL  
- **開発環境**: Docker
